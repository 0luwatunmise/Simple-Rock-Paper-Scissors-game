# Simple-Rock-Paper-Scissors-game
A simple Rock Paper Scissors game built for fun and learning.

# Features
Player chooses Rock, Paper, or Scissors

Computer randomly selects its move

Instant result display (Win / Lose / Tie)

Simple and interactive interface

# ⚙️ Installation & Usage
1. Clone this repository
   
   ```
   git clone https://github.com/0luwatunmise/Simple-Rock-Paper-Scissors-game.git
   ```

3. Navigate to the project folder

   ```
   cd Rock-Paper-Scissors
   ```

5. Open index.html in your browser

That’s it, You’re ready to play. Have FUN
